¿Qué es XML?

XML Extensible Markup Language es un lenguaje de marcado diseñado para almacenar y transportar datos.

No tiene etiquetas predefinidas (tú las creas).
Es muy estricto con la sintaxis.
Se usa para intercambiar información entre sistemas.

<persona>
  <nombre>Ana</nombre>
  <edad>25</edad>
</persona>

Aquí, tú decides cómo se llaman las etiquetas <persona>, <nombre>

¿Qué es XHTML?

XHTML Extensible HyperText Markup Language es una versión de HTML más estricta, basada en XML.

Se usa para crear páginas web.
Sigue las reglas estrictas de XML.
Todo debe estar bien cerrado y correctamente anidado.

<!DOCTYPE html>
<html>
  <head>
    <title>Mi página</title>
  </head>
  <body>
    <p>Hola mundo</p>
  </body>
</html>
