Introducción

El documento es una página web personal desarrollada en HTML ya añadiendo los estilos CSS integrados, que representa el portafolio digital. Su finalidad es mostrar información personal, intereses profesionales y trabajos, además de ofrecer una forma de contacto. En esta versión, se añade diseño visual básico mediante CSS, lo que mejora la presentación de la web y de cual quier web ya que son muy importantes en el momento de desarrollar una pagina web.

Aspectos técnicos

La página combina HTML para la estructura y CSS para el diseño, lo que la hace más completa y cercana a una web real.

Aspectos técnicos más importantes:

Metadatos en el <head>:
description, keywords y author para SEO.
viewport para diseño responsive.
Etiquetas Open Graph para compartir en redes sociales.
Favicon personalizado.
CSS interno <style>:
Uso de @font-face para cargar una fuente personalizada Ubuntu.
Estilo de encabezados h1, h2, h3 con color índigo.
Fondo general body en color GhostWhite.
Contenedores header, main, footer con:
Anchura fija de 600px.
Centrado automático margin: auto.
Fondo blanco.
Espaciado interno padding.
Estructura semántica HTML5:
<header> con navegación.
<nav> con enlaces internos.
<main> con diferentes secciones.
<section> para cada apartado.
<article> en el portafolio.
<footer> con enlaces externos.
Contenido multimedia y funcional:
Imágenes <img>.
Mapa incrustado iframe.
Formulario de contacto form, input, textarea.
Detalles a mejorar:
Etiqueta <footer> duplicada al final del documento.
ID duplicado en portafolio.
Algunas imágenes sin atributo alt.
Diseño con ancho fijo 600px poco adaptable a móviles aunque hay viewport.

¿Para qué sirve?

Esta página web sirve como portafolio personal online, permitiendo:

Presentar al autor y su perfil académico.
Mostrar sus áreas de interés desarrollo y diseño.
Visualizar posibles proyectos.
Facilitar el contacto con usuarios.
Tener presencia digital profesional.

Además, al incluir estilos CSS, también sirve como práctica para aprender diseño web básico.

Conclusión

En conclusión, esta versión mejora notablemente respecto a las anteriores vistas que teniamos acerca de la pagina web ahora al incorporar estilos CSS, lo que aporta una mejor presentación visual. La estructura HTML sigue siendo correcta y semántica, y se aplican buenas prácticas básicas de SEO. Sin embargo, es una base sólida para seguir evolucionando hacia una web más profesional y completa.
