Introducción

En este ejercicio he creado la estructura básica de un documento HTML. Este tipo de estructura es necesaria para poder desarrollar cualquier página web, ya que define el formato y la organización del contenido que se mostrará en el navegador.

 Aspectos técnicos

El documento comienza con la declaración <!doctype html>, que indica al navegador que se trata de un documento HTML5.
A continuación, se utiliza la etiqueta <html> con el atributo lang="es" para especificar que el idioma del contenido es español.

Dentro del <head> se incluyen:

<title>: define el título de la página que aparece en la pestaña del navegador.
<meta charset="utf-8">: establece la codificación de caracteres para permitir el uso de tildes y caracteres especiales.

Finalmente, en el <body> se define el contenido visible de la página, aunque en este caso está vacío.

 ¿Para qué sirve?

Esta estructura sirve como base para cualquier página web. Permite organizar correctamente el contenido y asegurar que el navegador interprete bien el código. Sin esta estructura, una página web podría no mostrarse correctamente o dar errores.

 Conclusión

Este ejercicio me ha ayudado a entender la importancia de la estructura básica en HTML. Es el primer paso para crear páginas web y es fundamental dominarlo antes de añadir contenido más complejo como texto, imágenes o estilos.
